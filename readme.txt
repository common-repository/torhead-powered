Implements Torhead and R2-DB JavaScript for tooltips.

== Description ==

If you want to enable Torhead and R2-DB tooltips on your site without editing files, this addon takes care of it all for you. Simply enable the plugin on your blog and any compatible link should work.

For more information on how to use tooltips for these sites, please checkout the following information:

* http://www.torhead.com/tooltips
* http://r2-db.com/guide/5/using-r2-db-com-tooltips-on-your-website

If you encounter a bug or have a development request then please let me know. You can reach me as follows:

* Website: http://www.manaobscura.com/torhead-wordpress-plugin/
* Email: tooltips@manaobscura.com
* Twitter: @gazimoff

Enjoy!

Based on the Powered by Wowhead plugin by nilved.net

== Installation ==

1. Upload `torhead-powered` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
